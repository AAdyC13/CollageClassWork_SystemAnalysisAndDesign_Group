# 專題題目：模組化記帳系統
#### 研究背景與動機
隨著智慧型手機的普及，記帳軟體成為許多人管理財務的重要工具。然而，現今市面上的記帳應用程式多半呈現兩極化的現象。
我們欲開發一款模組化記帳軟體，以「彈性」與「輕量」為核心理念。這款應用的基礎版本僅保留最基本的記帳功能，讓使用者能以最小的系統負擔完成日常記錄。而更進階的功能，則可透過插件和模組的形式自由擴充，讓使用者依個人需求打造專屬的記帳系統。
我們期望藉由這種設計，讓記帳軟體不再是固定框架的工具，而是一個能夠「使用者客製化」的財務管理平台。

#### 研究目的
1. **建立可運行的核心記帳系統**，具備「新增、編輯、刪除、查詢」等基本記帳功能的Android應用程式，確保能在Android 10以上系統穩定運作。
2. **實作模組化架構**，設計可動態載入與卸載功能模組的系統架構，使應用核心與插件之間以明確API介面溝通，達成用戶獨立開發與更新模組插件的目標。
3. **開發插件管理機制**，建立插件商店(私人伺服器內測)和模組管理介面，讓使用者能在應用內直接安裝、啟用、停用或移除插件。
